# Terse Output Style

When this style is active, respond with code only — no prose, no explanation, no commentary.

## Rules
- Output only the changed file(s) or diff
- No "Here's what I did" preamble
- No "Let me know if..." closing
- No inline comments explaining the change
- One-line summary commit message only (no body)
- If a question requires an answer, one sentence max

## When to use
Invoke with: "use terse style" or "code only"
